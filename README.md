# Pleiades_revisited
The full version of Table 4 from the Pleiades Binary Fraction Revisited paper is kept in the MRT [file](AAS70885_datafile4_Revision.txt). 

I thank Dr. Greg Schwarz (AAS Journals Data Editor) for his input on the file formatting.

See more about MRT (machine-readable table) standards: [https://journals.aas.org/mrt-standards/](https://journals.aas.org/mrt-standards/).

The paper is accepted for publication in the Astronomical journal and is available at: [http://arxiv.org/abs/2512.04143](http://arxiv.org/abs/2512.04143).
